Das Verb werden — Übungen

B1 · Die vier häufigsten Formen: werden · wird · wurde · würde

🔍 Schau dir diese Sätze an — was fällt dir auf?

Er wird langsam müde.

Das 🏠 Haus wurde komplett renoviert.

Ich würde gern Arzt werden.

Sie werden nächstes Jahr Eltern.

Das 💊 Medikament wird täglich verabreicht.

Er würde kommen, wenn er Zeit hätte.

Schritt 1: In welchen Sätzen beschreibt werden eine Veränderung — etwas verändert sich?

Schritt 2: „Das Haus wurde renoviert." — welche Funktion hat wurde hier?

Schritt 3: „Ich würde gern Arzt werden." — was drückt würde aus?

Schritt 4: Welche Form passt? „Das 📰 Ergebnis ___ morgen bekannt gegeben."

📋 Die vier wichtigsten Formen von werden

  • werden — Infinitiv · Präsens 1./3. Pl. · Futur I · Passiv Futur
  • wird — Präsens 3. Sg. · Vollverb (Zustandsveränderung) · Passiv Präsens
  • wurde — Präteritum 1./3. Sg. · Vollverb Vergangenheit · Passiv Präteritum
  • würde — Konjunktiv II · Wunsch · Hypothese · höfliche Bitte
Vollverb: Er wird Arzt. · Sie wurde krank. · Ich würde gern Lehrer werden.
Passiv: Das Haus wird gebaut. · Das Haus wurde gebaut.
Futur I: Wir werden das Problem lösen. · Du wirst es schaffen.
