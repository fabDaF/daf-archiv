# daf-archiv

Archiv für Alt-Ordner, die bei der Konsolidierung am 2026-04-10 aufgetaucht sind und nicht einfach gelöscht werden sollten, aber auch nirgendwo aktiv verwendet werden.

## Inhalt

- **B1 Grammatik Kopie/** – Alte Kopie des B1-Grammatik-Ordners
- **B2 Grammatik Kopie/** – Alte Kopie des B2-Grammatik-Ordners
- **Löschbar/** – Dateien, die als löschbar markiert, aber nicht gelöscht wurden

## Zweck

Sicherheitsanker. Sollte sich herausstellen, dass eine dieser Dateien doch noch gebraucht wird, ist sie hier versioniert abrufbar. Langfristig kann dieses Repo auch ohne Risiko gelöscht werden.
